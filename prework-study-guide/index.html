Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Use h1 for the main header, then h2,h3,etc..
  • Use ol for an ordered list.
  • Use ul for an unordered list.
  • Use li to list your items or text.
  • Use src to creat a link to a .png file.
  • Use alt to name the .png file.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • .card can apply rules to all sections with the class selection "card".(like a variable)
  • Using a comma(,) can apply rules to multiple HTML elements like "header","footer", and "h1".
  • We can add external CCS code to a HTML file using link rel "stylesheet" href="file-location".

Git

  • Use git status to check what branch we are currently on
  • Use git checkout -b branch-name to sreate a new branch and switch to it
