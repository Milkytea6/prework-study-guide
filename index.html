HTML

  • HTML is like the stucture of a website.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Use h1 for the main header, then h2,h3,etc..
  • Use ol for an ordered list.
  • Use ul for an unordered list.
  • Use li to list your items or text.
  • Use src to creat a link to a .png file.
  • Use alt to name the .png file.

CSS

  • CSS is like the style of a website.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • .card can apply rules to all sections with the class selection "card".(like a variable)
  • Using a comma(,) can apply rules to multiple HTML elements like "header","footer", and "h1".
  • We can add external CCS code to a HTML file using link rel "stylesheet" href="file-location".

Git

  • Git allows us to interact with repositories to create multiple verions of a project.
  • Use git status to check what branch we are currently on.
  • Use git checkout -b branch-name to sreate a new branch and switch to it.
  • Use cd to change directories.
  • Use mkdir to make a directory.
  • Use git checkout main to navigate to main branch.

Update repositoy on git hub to match local branch changes.

  1. Use git status to show current branch.
  2. Use git add -A to add the changes to working directory.
  3. Use git commit -m "message" to commit to repositoy with a message.
  4. Use git pull origin main to pull main branch and make sure it is up to date.
  5. Use git push origin feature/branch-name to push changes to github.
  6. Do the Pull request on github.
  7. Use git checkout main to navigate to main branch.
  8. Lastly use git pull origin main again to update the local branch with the remote branch.

JavaScript

  • JavaScript is like the functionality of a website.
  • A variable is a named container that allows us to store data in our code.
  • Make sure to not call a variable before it defined.
  • Control flow is the order in which a computer executes code in a script.
  • An if statement is a way of introducing decision-making into our code
  • An array is a single variable that is used to hold a group of data.
  • A for loop is used to repeat a code block over and over.
  • A for loop has 3 important statments:
    1. Declare a variable to begin a loop like let x = 0
    2. Then you need a condition so it knows how many times to repeat like x < array.length
    3. The final statement allows the array to iterate over each item like x+1 or x++
  • A function is packaging set of instructions that tells the computer how to perform a certain task.
  • Functions must be called in order to execute making it reusable.
  • An argument is a value (primitive or object) passed as input to a function.
  • The console.log() static method outputs a message to the console.
  • = will set a variable while === checks if two values are equal.
  • /*This text will not be executed in your code allowing notes and reminders.*/
  • //Works the same for one line of code.
